# Statify (in development)
This app shows statistics of your Spotify account using Spotify API.

You start by logging in.

<img width="1151" alt="Screenshot 2024-09-22 at 17 07 42" src="https://github.com/user-attachments/assets/ebc9057e-b3cd-44ca-9e57-783d32dde0d1">


Then you see the main page with all main stats from last 4 weeks, 6 months or lifetime; your profile's info, your subs and following on this app, etc.:

<img width="1125" alt="Screenshot 2024-09-22 at 17 09 27" src="https://github.com/user-attachments/assets/a01e6704-78ec-4a79-bcf2-949515248899">
